## Description

<hr/>

- 위치를 기반으로 서울시 와이파이 정보를 제공하는 서비스입니다.
- 원하는 위치나 혹은 현재 위치의 좌표를 이용하여 근처의 공공 와이파이 정보를 확인할 수 있습니다.

## Requirements

<hr/>

- JDK: JDK 1.8
- Server: Tomcat 8.5
